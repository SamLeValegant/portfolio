Recherche établissements par numéro - JavaScript

Script JavaScript permettant de rechercher des mairies et établissements scolaires à partir de numéros de téléphone, en utilisant des APIs publiques ouvertes.

Fonctionnalités principales

  • Recherche par numéro de téléphone
  • Intégration d'APIs publiques
  • Affichage des résultats en temps réel
  • Interface utilisateur simple
  • Gestion des erreurs
  • Cache local des résultats

Technologie utilisée

JavaScript HTML CSS APIs REST

À propos de ce projet

Ce projet m'a permis de découvrir comment consommer des APIs REST et travailler avec des données publiques. J'ai compris l'importance de la gestion d'erreurs et de l'expérience utilisateur.
